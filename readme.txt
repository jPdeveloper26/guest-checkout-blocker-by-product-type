Force login or registration for specific WooCommerce product types during checkout.

== Description ==

Guest Checkout Blocker by Product Type allows you to require customer registration or login for specific product types in WooCommerce. Perfect for digital downloads, services, or any products where you need customer accounts.

= Features =

* Force registration for specific product types
* Support for simple, variable, grouped, external, virtual, and downloadable products
* Customizable notice message for customers
* Easy-to-use settings interface
* Compatible with WooCommerce checkout process
* Multilingual ready
* Follows WordPress coding standards

= Use Cases =

* Require accounts for digital downloads
* Force registration for subscription products
* Mandate login for service-based products
* Control guest checkout on a per-product-type basis

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/guest-checkout-blocker-by-product-type` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WooCommerce > Guest Checkout Blocker to configure the plugin
4. Select which product types should require registration
5. Customize the message shown to guests
6. Save your settings

== Frequently Asked Questions ==

= Does this plugin work with all WooCommerce product types? =

Yes, the plugin supports all standard WooCommerce product types and can be extended to support custom product types through filters.

= Can I customize the message shown to guests? =

Yes, you can customize the message through the plugin settings page under WooCommerce > Guest Checkout Blocker.

= Will this affect existing orders? =

No, this plugin only affects new checkout sessions. Existing orders and customers are not affected.

= Can I exclude specific products within a product type? =

Currently, the plugin works on a product type basis. For more granular control, you can use the provided filters in your theme or custom plugin.

= Is the plugin compatible with WooCommerce Subscriptions? =

Yes, if you have WooCommerce Subscriptions installed, you can force registration for subscription products.
